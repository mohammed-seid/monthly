## Color Scheme

### Primary Colors
- **Header/Navbar**: White background with OAF green border
- **Headings**: OAF green (#2b7f68)
- **Accents**: OAF gold (#e8b757)
- **Borders**: Light green (#e8f4f0)

### Text Colors
- **Primary Text**: Dark gray (#111111)
- **Secondary Text**: Medium gray (#333333)
- **Muted Text**: Light gray (#666666)

### Background Colors
- **Page Background**: Light gray (#f7f7f7)
- **Content Background**: White
- **Callout Background**: Light green (#e8f4f0)

## Responsive Design

### Desktop (760px+)
- 2-column grid for newsletter listings
- Full-width content
- Sidebar navigation
- Optimized spacing

### Mobile (<760px)
- 1-column grid for newsletter listings
- Full-width content
- Stacked navigation
- Optimized touch targets

## Interactive Elements

### Hover Effects
- Newsletter cards: Border color changes to OAF green, slight lift effect
- Links: Color changes to OAF gold
- Buttons: Background changes to OAF gold

### Buttons
- **Subscribe Button**: OAF gold background with dark text
- **Read More**: OAF green text with underline on hover

## Typography

### Fonts
- **Font Family**: System UI (Segoe UI, Roboto, Helvetica Neue, Arial)
- **Fallback**: Sans-serif

### Font Sizes
- **H1 (Page Title)**: 2rem
- **H2 (Section Title)**: 1.5rem
- **H3 (Subsection)**: 1.25rem
- **Body Text**: 1rem
- **Small Text**: 0.875rem

### Font Weights
- **Regular**: 400
- **Bold**: 700
- **Masthead**: 700

## Navigation

### Navbar Items
- **Logo**: OAF Ethiopia logo (clickable, links to homepage)
- **Title**: "OAF Ethiopia MEL"
- **Newsletter Link**: Links to homepage
- **GitHub Icon**: Links to GitHub repository
- **Twitter Icon**: Links to Twitter account

### Footer Links
- **Subscribe Button**: Links to email subscription
- **Copyright**: © One Acre Fund Ethiopia — MEL

## Accessibility Features

- Semantic HTML structure
- Proper heading hierarchy
- Alt text for images
- Color contrast compliance
- Keyboard navigation support
- Mobile-friendly design

## Performance Optimizations

- Minified CSS
- Optimized images
- Cached computations (freeze: true)
- Fast page load times
- Responsive images

## SEO Features

- Proper meta tags
- Semantic HTML
- Descriptive titles
- Structured content
- Mobile-friendly design

This structure provides a professional, branded experience for OAF Ethiopia's monthly newsletter while maintaining ease of use and accessibility.
